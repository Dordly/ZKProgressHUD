A easy-to-use HUD for your iOS app.

## Requirements

- iOS 8.0+
- Xcode 8+
- Swift 3.0+

## Installation

### CocoaPods

You can use [CocoaPods](http://cocoapods.org/) to install `ZKProgressHUD`by adding it to your `Podfile`:

```ogdl
platform :ios, '8.0'
use_frameworks!

target 'MyApp' do
pod 'ZKProgressHUD'
end
```
